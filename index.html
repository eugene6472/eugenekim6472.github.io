Project(Android Studio)

Android Studio I am currently developing an android game that will soon be released in the Google store. The game, called Connect Three, follows the rules of tic-tac-toe, shows user location, allows desktop play, and automatically determines the winner. User statistics like game time and number of victories are delivered to a server that uses MySQLS to store the information, and game results are delivered from the client using optimized tables. In the next version, I aim to prevent data loss and reduce unnecessary HTTP communication overhead through optimization of server communication.
